Documentation

Integration is a thin wrapper at the tool-call boundary. Target: a stranger integrates against one sandbox in under 30 minutes.

quickstart · register → dispatch → reconcile
const effect_id = await detent.registerIntent({
  stable_ids: { order_id: "ORD-88213", checkout_id: "C-88213" },
  effect_type: "order.create", effect_class: "IRREVERSIBLE",
  scope: "shop:acme", authority_ref: "authz_9c31", stamped_at: "…"
});
// idempotency key derives ONLY from operation_id — never model output
const receipt = await detent.dispatch(effect_id);
if (receipt === "AMBIGUOUS") await detent.reconcile(effect_id);

No universal exactly-once (Two Generals / FLP). True rollback of an externalized effect is impossible — only compensation, itself a new effect with its own failure modes, measured never assumed. The LLM never holds sole authority over an irreversible action.

Responsive behavior

  • Desktop-first at 1440×960, polished to 1280px and laptop widths.
  • Sidebar collapses to a 60px icon rail below ~1120px; labels hide, icons remain.
  • Three-region investigation reflows: 3-col → left+center (right wraps under) → fully stacked evidence panels.
  • Dense tables scroll horizontally with a sticky header; secondary detail moves into the expandable row.
  • No fake mobile app — this is a desktop investigation tool.

Accessibility

  • WCAG AA contrast in both themes; status carried by icon + text, never color alone.
  • Full keyboard model: ⌘K palette, j/k row nav, ↵ to open, esc to close.
  • Visible focus rings on every interactive element (2px accent, offset).
  • Reduced-motion honored — all transitions collapse to near-zero.
  • Accessible dialogs, drawers, tooltips (labels, roles, focus trap in impl.).

# Licensing

**Copyright (c) 2026 the repository owner. All rights reserved.**

No LICENSE file exists, deliberately. This repository is private and sole-authored; under
default copyright law no one may reproduce, distribute, or create derivative works from it.
That is the correct maximal-optionality state for a private, pre-release repository: adding a
provisional license now would grant nothing useful and would pre-commit the narrative before
the decision is made.

- The licensing decision is **OPEN** — see
  [ADR-0014](docs/decisions/0014-licensing.md) for the options analysis, one-way doors,
  precedents, and current leaning (Apache-2.0 for code, CC-BY-4.0 for standalone dataset/doc
  artifacts, DCO for inbound — decided at the public-release gate, not before).
- **No contributions are accepted** while this record is open. Do not merge outside work of
  any kind.
- The decision must close **before the first public release** (see the public-release gate in
  [docs/execution-plan.md](docs/execution-plan.md)).
